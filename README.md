# SoundCloud Plugin for Nuclear

A plugin that provides metadata and streaming from [SoundCloud](https://soundcloud.com/). Search for tracks and artists, then play them directly via HLS streaming.

## Installation

Install from the Nuclear plugin browser.

## How It Works

This plugin extracts a client ID from SoundCloud's public website and uses it to access the internal v2 API. No registered app or API key is required. Audio is streamed via HLS (AAC).
